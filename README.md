# Analysis-of-variance
 Using multiple statistical tests to analyze precipitation data between Buffalo, NY and San Diego, CA.
 Executing ANOVA, Kruskal-Wallis, and Meidan tests to analyze and test hypotheses of the annual precipitation in Buffalo, NY and San Diego, CA in 3 time periodds over the course of 30 years using random samples.
 Evidence shows that the tests conducted, through 6 step hypothesis testing, that certain tests can be better suited to different data, especially between both cities.
 Used R, Rsstudio to execute the script.
 This assignment is useful for comparing the data between the cities, but more data might be needed to make assumptions about subjects such as climate change.
